@Depack/Form

\n\nWelcome to the website about @depack/form component that allows to build Bootstrap forms via the Preact library.\n\n

\n \n \n \n

\n\n

Writing Forms In JSX

\n\nThe main advantage of this package is that it allows to write forms in JSX language easily, using the same structure as Bootstrap form groups, but with the FormGroup component automatically generating the id and help-id for inputs.\n\n
import Form, { FormGroup, TextArea, Input, Select } from '../src'\n\nconst ExampleForm = ({ ...props }) => (<Form {...props}>\n  <FormGroup label=\"Input\" help=\"Type in something...\">\n    <Input name=\"input\" value=\"hello-world\" />\n  </FormGroup>\n  <FormGroup label=\"Select\" help=\"Please select...\">\n    <Select name=\"select\" value=\"2\" options={[\n      {\n        title: 'Free will',\n        value: '1',\n      },\n      {\n        title: 'Unfree will',\n        value: '2',\n      },\n    ]} />\n  </FormGroup>\n  <FormGroup label=\"TextArea\" help=\"Multiple row input...\">\n    <TextArea name=\"textarea\">\n      One must still have chaos in oneself to be able to give birth to a dancing star.\n    </TextArea>\n  </FormGroup>\n</Form>)\n\nexport default ExampleForm
\n\n

\n \n \n \n

API

\n\nThe package exports multiple components:\n\n
import Form, {\n  FormGroup, Input, TextArea, Select,\n} from '@depack/form'
\n\nRead about each in the GitHub documentation.\n\n
